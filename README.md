# Mindmeld

Multiplayer party games that bring people together through shared thinking.

## Stack

- **Go** - Server
- **templ** - Type-safe HTML templates
- **HTMX** - Frontend interactivity (coming soon)
- **Tailwind CSS** - Styling
- **PostgreSQL** - Database
- **Air** - Hot reload for development

## Prerequisites

- Go 1.25+
- Docker (for PostgreSQL)
- [Tailwind CSS CLI](https://github.com/tailwindlabs/tailwindcss/releases) (standalone binary)

### Install Tailwind CLI (macOS ARM)

```bash
curl -sLO https://github.com/tailwindlabs/tailwindcss/releases/latest/download/tailwindcss-macos-arm64
chmod +x tailwindcss-macos-arm64
sudo mv tailwindcss-macos-arm64 /usr/local/bin/tailwindcss
```

## Getting Started

```bash
# Copy environment config
cp .env.example .env.local

# Start PostgreSQL
make db-up

# Build CSS
make css

# Start dev server with hot reload
make dev
```

Open [http://localhost:3000](http://localhost:3000) to see the app.

## Available Commands

| Command | Description |
|---------|-------------|
| `make dev` | Start server with hot reload |
| `make run` | Run server directly |
| `make build` | Build production binary |
| `make templ` | Generate templ templates |
| `make css` | Build Tailwind CSS |
| `make css-watch` | Watch and rebuild CSS |
| `make db-up` | Start PostgreSQL container |
| `make db-down` | Stop PostgreSQL container |
| `make clean` | Remove build artifacts |

## Deployment

### Digital Ocean App Platform

This app deploys to DO App Platform using auto-detection (no Dockerfile or spec file needed).

**Requirements for auto-detection:**
- Valid `go.mod` at repo root
- Single service (`cmd/server/main.go`)
- App listens on `$PORT` env var

**Generated files must be committed:**

The following generated files are committed to git so the production build only needs `go build`:

- `templates/*_templ.go` - Generated by templ
- `internal/db/*.go` - Generated by sqlc
- `static/css/output.css` - Generated by Tailwind

If you modify templates, queries, or styles, regenerate and commit:
```bash
make generate  # templ + sqlc
make css       # tailwind
```

**DO App Platform settings:**
- Build command: `go build -o bin/server ./cmd/server`
- Run command: `bin/server`
- HTTP Port: `8080`

**Environment variables to set in DO:**
- `DATABASE_URL` - Connection string (auto-injected if using managed Postgres)

**Migrations:**

Run migrations against production DB after deploy:
```bash
DATABASE_URL="your-prod-url" make migrate
```
